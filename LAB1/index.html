Gestión de Flota de Vehículos Eléctricos

Plataforma de movilidad sostenible

Aplicaciones Web para una gestión eficiente de vehículos eléctricos en el Grado en Ingeniería del Software de la Facultad de Informática.

Beneficios de los vehículos eléctricos

  1. Reducción de emisiones contaminantes
  2. Menor coste de mantenimiento
  3. Mayor eficiencia energética

Servicios principales de la plataforma

  • Gestión de rutas
  • Monitoreo en tiempo real
  • Optimización de carga

Definiciones clave

HTML
Lenguaje de marcado para estructurar contenido web.
CSS
Lenguaje de estilos para diseñar la apariencia de las páginas web.
JSON
Formato ligero de intercambio de datos.
